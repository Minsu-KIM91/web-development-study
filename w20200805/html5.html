+ HTML5 - 웹 어플리케이션: 웹 자체가 하나의 애플리케이션 처럼 작동하는 것. (html5로 만들면 이런 기능이 가능하다.) ex) pdf문서를 볼 때도 리더기가 있어야 했는데 이제는 그냥 크롬창열어서 바로 볼 수 있는 것을 이야기함. -웹 기능은 점점 확대 발전하는데 웹 문서를 만드는 HTML4는 기능이 매우 제한적 HTML4에서는 텍스트, 이미지, 표 등을 삽이바는 기능 정도만 있음. HTML4에서 다양한 기능을 구현하려면 ActiveX, Flash, Ajax 등 프로그래밍으로 구현해야 함. cf) 네이티브 어플리케이션 : 컴퓨터에 서리한 후 실행해야 하는 프로그램 ex) 포토샵 - 직접 프로그램을 설치해서 사용하는 것을 의미한다. -> 이젠 웹에서 그냥 사용이 가능해짐.
